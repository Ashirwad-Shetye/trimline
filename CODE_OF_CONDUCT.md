# Code of Conduct

Trimline is an open-source project for readers, researchers, and contributors who want a calmer web without hostile or invasive behavior.

## Expected behavior

- Be respectful and specific.
- Assume good intent, but accept corrections.
- Keep feedback focused on the project, code, documentation, and user experience.
- Help make issues reproducible with URLs, screenshots, browser version, and steps.

## Unacceptable behavior

- Harassment, insults, threats, or discriminatory language.
- Posting private information without permission.
- Repeated low-effort comments after maintainers ask for more detail.
- Pushing changes that intentionally weaken user privacy or control.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block participants; or reject contributions that violate this code. If you see a problem, open a private maintainer contact path once one is available, or flag it through the hosting platform.

## Project values

Trimline is local-first, reversible, conservative, and reader-focused. Contributions should preserve those values.
